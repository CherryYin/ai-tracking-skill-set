## 📦 包含的 Skill

### 1️⃣ ai-daily-news（AI 日报）
**功能**: 抓取 AI 热点新闻 + 论文 + 图片 + 自动生成文章

**位置**: `ai-daily-news/`

**使用**:
```bash
cd ai-daily-news/scripts
python3 fetch_ai_daily.py \
  --hn-limit 5 \
  --arxiv-limit 5 \
  --download-images \
  --output output/ai_daily_data.json
```

**参数**:
- `--hn-limit`: HackerNews 条目数
- `--arxiv-limit`: ArXiv 论文数
- `--download-images`: 下载图片
- `--date`: 指定日期
- `--no-article`: 不生成文章

### 2️⃣ financial-analysis（财经分析）
**功能**: 财经新闻 + 个股数据（TuShare / AKShare）

**位置**: `financial-analysis/`

**使用**:
```bash
# 获取新闻
cd financial-analysis/scripts
python3 fetch_news.py --limit 100 --token YOUR_TUSHARE_TOKEN

# 获取行情
python3 fetch_stock_data.py --stock_code 600519 --days 7 --token YOUR_TUSHARE_TOKEN
```

**依赖**:
- TuShare Token（必需）
- 或使用 AKShare（可选）

### 3️⃣ github-hot-repos（GitHub 热门仓库）
**功能**: 搜索热门 GitHub 仓库 + 抓取 README

**位置**: `github_hot_repo_collect_analysis/github-hot-repos/`

**使用**:
```bash
cd github_hot_repo_collect_analysis/github-hot-repos/scripts
python3 search_github_repos.py \
  --start-date "30 days ago" \
  --end-date "1 days ago" \
  --query "llm" \
  --sort stars \
  --limit 20
```

**依赖**:
- GitHub PAT（环境变量）
