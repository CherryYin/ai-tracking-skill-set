## 总体结构

文章应按照以下顺序组织：

```markdown
# 文章标题

## 导语段落
（100-200字的引入）

## [序号] [子标题]
内容描述
图片（如果有）
原文链接

## [序号] [子标题]
...

## 结语
（100-200字的总结）
```

## 标题规范

### 文章主标题
- **要求**：吸引人、概括性强、包含日期
- **格式**：`# AI日报：[日期] - [核心亮点]`
- **示例**：
  - `# AI日报：2024-01-15 - GPT-5发布预览版`
  - `# AI日报：2024-01-16 - 多模态AI迎来新突破`

### 子标题
- **要求**：简洁明了，直接点出内容核心
- **格式**：`## [序号] [核心内容]`
- **序号**：使用数字1-10
- **示例**：
  - `## 1. OpenAI发布GPT-5预览版`
  - `## 2. DeepMind新算法提升RL训练效率`
  - `## 3. Anthropic推出AI安全评估框架`

## 内容要求

### 导语段落
- **位置**：主标题之后，第一条内容之前
- **长度**：100-200字
- **内容**：
  - 概括当天AI领域的主要亮点
  - 提及2-3个最值得关注的方向
  - 吸引读者继续阅读

**示例**：
```markdown
## 今日AI要闻

今天AI领域迎来了多项重要进展，OpenAI发布了备受期待的GPT-5预览版，DeepMind推出了更高效的强化学习算法，同时多篇关于大模型安全性的论文引发广泛讨论。以下是精选的10条重要消息。
```

### 正文条目
每条内容应包含：

1. **核心描述**（必需）
   - 长度：150-300字
   - 内容：
     - 清晰说明这是什么消息
     - 解释为什么重要/值得关注
     - 提供关键数据或细节
     - 用通俗易懂的语言解释技术概念

2. **图片**（可选，推荐）
   - 位置：核心描述之后
   - 数量：每条1-2张
   - 选择原则：
     - 优先选择架构图、流程图
     - 其次选择实验结果图、产品截图
     - 避免选择装饰性图片

3. **要点列表**（推荐）
   - 当消息包含多个关键点时，使用列表形式
   - 格式：使用 `-` 或数字列表

4. **原文链接**（必需）
   - 格式：`[阅读原文](URL)`
   - 位置：条目末尾

**示例**：
```markdown
## 1. OpenAI发布GPT-5预览版

OpenAI今日正式发布了GPT-5的预览版本，在推理能力和多模态理解方面取得重大突破。根据官方介绍，新模型在MMLU基准测试中得分达到89.5%，比GPT-4提升了15个百分点，展现出更强的逻辑推理能力。

![GPT-5架构图](./ai_daily_articles/images/01_gpt5_arch.png)

主要改进包括：
- 增强的逻辑推理能力，数学问题求解准确率提升40%
- 更精准的多模态理解，支持图像、视频、音频统一处理
- 显著降低了幻觉产生率，事实准确性提高
- 支持更长上下文（最大128K tokens），适合处理长文档

该模型已在ChatGPT Plus用户中开始逐步开放测试，预计将在下月全面上线企业版API。

[阅读原文](https://openai.com/blog/gpt5-preview)
```

### 结语段落
- **位置**：所有10条内容之后
- **长度**：100-200字
- **内容**：
  - 总结当天AI发展的总体趋势
  - 提出有价值的观察或预测
  - 引导读者关注未来方向

**示例**：
```markdown
## 总结

今天的AI发展呈现出几个明显趋势：大型语言模型继续向更强的推理能力演进，多模态融合成为技术发展的主要方向，同时AI安全和可解释性也受到越来越多关注。随着技术的快速进步，我们预计未来几个月将有更多实用的AI应用落地。
```

## 图片规范

### 图片选择原则
1. **信息密度优先**
   - 架构图 > 流程图 > 实验结果图 > 产品截图 > 概念图

2. **清晰度要求**
   - 分辨率：至少800x600像素
   - 文字清晰可读
   - 关键信息不模糊

3. **相关性**
   - 图片必须直接服务于内容理解
   - 避免纯装饰性图片

### 图片插入格式
```markdown
![图片描述](./ai_daily_articles/images/文件名.扩展名)
```

**说明**：
- `图片描述`：简短的图片说明，5-20字
- `文件名`：`{序号}_{类型}.{扩展名}`，如 `01_architecture.png`
- `扩展名`：jpg、png、gif、webp

### 图片描述示例
- `![GPT-5架构图](./ai_daily_articles/images/01_gpt5_arch.png)`
- `![实验结果对比](./ai_daily_articles/images/02_results.jpg)`
- `![系统流程示意图](./ai_daily_articles/images/03_workflow.png)`

### 图注格式（可选）
如果图片需要详细说明，可以在下方添加图注：
```markdown
![GPT-5架构图](./ai_daily_articles/images/01_gpt5_arch.png)

*图1：GPT-5的整体架构，包含改进的多头注意力机制和跨模态融合模块*
```

## 链接格式

### 原文链接
```markdown
[阅读原文](URL)
```

### 内部引用
如果文章中需要引用之前提到的内容：
```markdown
（参见上文第2条）
（详见第5条的消息）
```

### 引用数据来源
如果引用了具体数据，建议注明来源：
```markdown
根据OpenAI官方发布的数据...
根据ArXiv论文的实验结果...
```

## 其他规范

### 语言风格
- 使用中文撰写
- 专业术语首次出现时提供简单解释
- 避免过度使用技术缩写（如：MLP、CNN、BERT等，首次使用时写出全称）
- 保持客观中立的语气

### 段落组织
- 每段不超过200字
- 使用空行分隔不同主题
- 重要信息放在段落开头

### 特殊情况处理

#### 无图片的内容
如果某条内容没有合适的图片：
```markdown
## 7. Meta发布AI伦理研究白皮书

Meta今日发布了一份关于AI伦理的研究白皮书，提出了建立AI伦理评估框架的建议。该框架包含...

（注：该消息暂无相关配图）

[阅读原文](https://...)
```

#### 图片下载失败
如果图片下载失败，在文章中说明：
```markdown
![配图获取失败](./ai_daily_articles/images/placeholder.png)

*注：原图片无法获取，建议读者访问原文查看*
```

#### 内容过于相似
如果多条内容涉及同一事件，在导语中说明：
```markdown
## 今日AI要闻

今天AI领域最引人注目的是关于GPT-5的多项动态，包括官方发布、社区讨论和技术分析等。以下是相关内容的详细报道。
```
