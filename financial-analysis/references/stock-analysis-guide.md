# 股票分析框架

## 概览
本指南提供个股深度分析的多维度框架，结合新闻和行情数据进行综合评估。

## 分析维度

### 1. 盈利能力分析

#### 1.1 财务指标
- 营业收入及增长率
- 净利润及增长率
- 毛利率、净利率
- ROE（净资产收益率）
- ROA（总资产收益率）

#### 1.2 盈利质量
- 现金流状况：经营现金流、自由现金流
- 盈利持续性：主营业务占比
- 非经常性损益影响

#### 1.3 历史对比
- 近3年盈利趋势
- 同行业对比
- 季度环比变化

### 2. 自身潜力分析

#### 2.1 成长性
- 营收增长驱动力
- 新产品/新市场拓展
- 技术创新能力
- 品牌影响力

#### 2.2 竞争优势
- 护城河分析：技术、品牌、渠道、成本
- 市场地位：行业排名、市场份额
- 核心团队：管理能力、执行力

#### 2.3 研发投入
- 研发费用占营收比例
- 专利储备
- 技术迭代能力

### 3. 行业潜力分析

#### 3.1 行业景气度
- 行业生命周期阶段
- 行业增长率
- 政策支持力度

#### 3.2 市场空间
- 市场规模及增长潜力
- 细分领域机会
- 国际化可能性

#### 3.3 产业链位置
- 上游：原材料、供应商议价能力
- 中游：制造、加工、技术壁垒
- 下游：客户、销售渠道、终端市场

### 4. 主要形势分析

#### 4.1 技术面
- 价格趋势：上升、下降、震荡
- 成交量变化
- 技术指标：MA、MACD、RSI等
- 支撑位与压力位

#### 4.2 市场情绪
- 新闻情感分析：正面、负面、中性
- 研报评级
- 机构调研频次
- 投资者关注度

#### 4.3 风险因素
- 系统性风险：市场整体风险
- 特质性风险：公司自身风险
- 行业风险：政策、竞争、技术变革
- 财务风险：债务、流动性

### 5. 价值与购买代价分析

#### 5.1 估值水平
- PE（市盈率）：当前、历史、行业对比
- PB（市净率）：当前、历史、行业对比
- PS（市销率）：适用于高增长公司
- PEG：考虑增长率的估值指标
- DCF估值：现金流折现法

#### 5.2 安全边际
- 当前价格 vs 内在价值
- 下跌空间评估
- 最佳买入时机判断

#### 5.3 配置建议
- 投资比例：根据风险承受能力
- 持有期限：短期、中期、长期
- 操作策略：分批建仓、定投、一次性买入

## 分析步骤

### 步骤1：数据收集
- 获取股票历史行情数据（最近一周、一月、三月）
- 收集相关新闻和公告
- 获取最新财务数据（若有）

### 步骤2：多维度评估
- 盈利能力：打分 1-10
- 自身潜力：打分 1-10
- 行业潜力：打分 1-10
- 当前形势：打分 1-10
- 估值水平：打分 1-10

### 步骤3：综合评分
- 计算综合得分
- 识别优势与短板
- 定位投资价值

### 步骤4：风险提示
- 列出主要风险点
- 评估风险等级
- 提供应对建议

## 输出格式

### 个股分析报告结构
```markdown
## 个股深度分析：[股票名称] ([股票代码])

### 1. 盈利能力分析
- 核心指标
- 历史趋势
- 行业对比

### 2. 自身潜力分析
- 成长性评估
- 竞争优势
- 研发创新

### 3. 行业潜力分析
- 行业景气度
- 市场空间
- 产业链地位

### 4. 主要形势分析
- 技术面分析
- 市场情绪
- 风险因素

### 5. 价值与购买代价分析
- 估值评估
- 安全边际
- 配置建议

### 6. 综合评估
- 综合得分：X/10
- 投资评级：强烈推荐/推荐/中性/谨慎/回避
- 核心观点：3-5句话总结

### 7. 风险提示
- 主要风险点
- 应对建议
```

## 注意事项

1. 数据时效性：确保使用最新的财务和行情数据
2. 信息全面性：多角度分析，避免单一指标误导
3. 客观中立：基于数据和事实，避免情绪化判断
4. 风险意识：任何投资都有风险，充分提示风险因素
5. 投资有风险，入市需谨慎：本分析仅供参考，不构成投资建议
