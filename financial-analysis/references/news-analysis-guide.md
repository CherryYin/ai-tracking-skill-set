# 新闻分析指导

## 概览
本指南提供财经新闻分析的标准化框架，帮助从海量新闻中提取有价值的信息。

## 分析维度

### 1. 宏观形势分析

#### 1.1 政策导向
- 识别政策关键词：货币、财政、产业、监管等
- 政策力度判断：鼓励、限制、中性
- 影响范围评估：全行业、特定板块、个别公司

#### 1.2 经济指标
- GDP、CPI、PMI等关键指标的提及
- 趋势判断：上升、下降、持平
- 预期影响：利好、利空、中性

#### 1.3 行业趋势
- 热门行业识别：人工智能、新能源、半导体等
- 行业生命周期判断：成长期、成熟期、衰退期
- 产业链位置：上游、中游、下游

### 2. 热门个股特点分析

#### 2.1 识别方法
- 频次统计：新闻中出现频率最高的公司
- 情感分析：正面、负面、中性报道比例
- 时间集中度：短时间内大量相关新闻

#### 2.2 特征提取
- 公司类型：科技、金融、消费、制造等
- 市值规模：大盘股、中盘股、小盘股
- 概念标签：龙头、概念股、ST等

#### 2.3 事件关联
- 财报相关：业绩预告、年报、季报
- 重大事项：并购重组、股权变更、产品发布
- 监管动态：问询函、处罚、警示

## 分析步骤

### 步骤1：数据清洗
- 去除重复新闻
- 过滤广告和无关内容
- 统一时间格式

### 步骤2：分类标记
- 按主题分类：宏观、行业、个股
- 按情感标记：正面、负面、中性
- 按重要性排序：头条、重要、普通

### 步骤3：趋势提取
- 统计高频词汇
- 识别热点话题
- 发现潜在趋势

### 步骤4：综合判断
- 多源验证：不同新闻源的同一事件
- 交叉分析：政策与市场的关联
- 风险提示：潜在的负面因素

## 输出格式

### 宏观形势总结
```markdown
## 宏观形势分析

### 政策环境
- 主要政策动向：...
- 政策影响：...

### 经济指标
- 关键数据：...
- 趋势判断：...

### 行业热点
- 热门行业：...
- 产业趋势：...
```

### 热门个股总结
```markdown
## 热门个股分析

### 前五大热门股
1. 股票名称 (代码)
   - 新闻频次：X次
   - 主要事件：...
   - 市场表现：...

### 热点概念
- 概念名称：相关股票...
- 驱动因素：...
```

## 注意事项

1. 时效性：财经新闻时效性强，注意发布时间
2. 权威性：优先关注官方媒体和权威机构发布
3. 全面性：避免单一信息源，多方验证
4. 客观性：区分事实报道与观点评论
5. 风险意识：识别可能的利益相关方和炒作行为
